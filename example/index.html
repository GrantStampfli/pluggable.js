Pluggable.js example

Here's a small example which shows to use pluggable.js to make a private, closured Javascript object pluggable.

There are two Javasript files included in this page, app.js and plugin.js. Fee free to open them up and see how they look inside. The code has explanatory comments and is also explained in the README of pluggable.js.

In app.js, there is a private object called _private, with a method showNotification, which cannot be accessed by code outside of that private object.

However, since this private object uses pluggable.js, we can now write a plugin which does override this method, even though it's closured. The plugin code is contained in plugin.js.
